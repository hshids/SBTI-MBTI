# SBTI-MBTI 匹配系统：从死板贴标签到动态概率云的奇妙进化
引言：我们为什么要改掉原先的算法？

最近 SBTI (自我基础特征测试) 在小红书上真的超级火！很多人（包括我们自己）第一反应就是想把它和经典的 MBTI 对号入座，比如觉得“CTRL(拿捏者)”肯定就是 ENTJ，“SOLO(独行者)”绝对是 INTJ。

但是，看了大家真实的测试结果后，我们被疯狂打脸了！我们发现，同一个 MBTI 类型（比如 INFP），居然能在 SBTI 里测出几十种完全不同的结果。这就逼着我们重新去琢磨这两个测试底层的逻辑到底有啥不一样。最后，我们决定彻底改变思路：放弃那种死板的“一对一”匹配，升级成更好玩、也更符合人性的“四象限和概率云”模式！

一、 它们到底是怎么测的？两把“尺子”的错位

要想搞懂怎么对应，咱们得先看看这两个测试是怎么“看人”的。

![系统概览：底层维度的对比](Screenshot 2026-04-09 at 1.24.51 PM.jpg)

1. SBTI 测的是啥？捕捉你的“当下状态”

说白了，SBTI 就像是你当前生活状态的快照，它看的是你这段时间里的：

S维（自我认知）： 你最近自信吗？清楚自己要什么吗？

E维（情感模式）： 你在关系里有安全感吗？投入吗？边界感强不强？

A维（世界观）： 你觉得世界美好吗？喜欢守规矩还是瞎折腾？

Ac维（行动力）： 你做事有动力吗？果断吗？还是喜欢临时抱佛脚？

So维（社交风格）： 你最近喜欢社交吗？平时爱戴面具还是做真实的自己？

2. MBTI 测的是啥？你的“底层出厂设置”

相比之下，MBTI 测的更像是一个人长期的底层习惯：

E/I（能量方向）： 你是靠热闹充电，还是靠独处回血？

S/N（信息处理）： 你看重眼前的细节，还是更关注未来的大趋势？

T/F（决策方式）： 你做决定是讲究客观逻辑，还是凭感情和价值观？

J/P（生活结构）： 你喜欢按计划办事，还是走到哪算哪？

简单总结一下： SBTI 测的是**“你最近这段时间的心累程度和生活状态”（就像个随时会换的面具），而 MBTI 测的是“你骨子里的性格底色”**（你的出厂设置）。要是硬拿性格底色去对号入座某一个状态面具，那肯定会出大错的！

二、 我们是怎么发现规律的？算法的打脸与进化

1. 以前的坑：强行“对号入座”行不通

一开始，我们确实想过直接换算（比如觉得喜欢社交肯定就是 E，行动力强就是 J）。但看了大家真实的反馈后，我们发现了几个特别不按套路出牌的现象：

现象 A：INFP 简直是“状态收集器”！ INFP 不仅测出了 LOVE-R(多情者)，还大把大把地测出 JOKE-R(小丑)、FUCK(草者)、MALO(吗喽) 甚至是 IMSB(傻者)。这跨度也太大了！

现象 B：E 和 I 居然反转了？ 像 FUCK(草者) 和 WOC!(握草人) 听起来是不是很外向、很暴躁？但实际上，测出这两个结果的人里，有非常多是内向的 I 人！

2. 发现新大陆：隐藏的“四象限”

为了搞清楚大家为什么会测出这些看似矛盾的结果，我们仔细分析了一下数据，终于发现了 SBTI 背后隐藏的真实坐标系（它其实根本不是 MBTI 的那四个字母哦）：

![四象限架构解析](Screenshot 2026-04-09 at 1.24.31 PM.jpg)

Y轴（掌控推进度）： 就是你的自律程度、执行力和对秩序的渴望。

X轴（外向介入度）： 重点来了，这就是解开谜题的钥匙！ 这里的“外向”可不是指你是不是个社交达人，而是指你内耗大不大、投入的情绪和精力多不多。

这就完美解释了为什么 INFP 会测出 JOKE-R(小丑) 这些**“第四象限（卷入波动型）”**的结果啦！因为当 INFP 陷入精神内耗或者特别共情的时候，他们对周遭人和事的情感投入非常高。所以，在测试里他们就被判定为右侧的“高介入”，看起来就像是变成了外向的人一样！

3. 最终大招：超直观的“双向概率热力图”

有了这些发现，我们彻底抛弃了那种死板的表格对应，做出了一个全新的双向概率云系统！我们现在用“主阵地（最容易出的结果）”和“游离区（压力大时容易出的结果）”来展示。

是不是超级直观？而且不管是从数学逻辑还是心理学角度看，都非常合理哦！

三、 挖挖背后的心理学：SBTI 为什么这么戳人？

1. 心理学依据：特质-状态理论

这其实和心理学上的“特质-状态焦虑理论”不谋而合！
简单来说，MBTI 就像是你的特质 (Trait)，是很稳定的性格底色；而 SBTI 非常像你的状态 (State)，是你面对生活压力和挫折时做出的即时反应。
我们的热力图，其实画的就是**“不同性格的人，在遇到现代生活的各种毒打时，大概率会变成什么样”**。

2. 大家都在想什么？SBTI 爆火背后的社会心理

为什么像 MALO(吗喽)、DEAD(死者)、POOR(贫困者)、FUCK(草者) 这些听起来有点自嘲、甚至有点攻击性的人格，会让大家觉得那么准呢？

压力下的自我保护： 像 FAKE(伪人) 和 JOKE-R(小丑) 出现得特别多，说明大家在现在这么高压的社交和职场环境里，不得不发展出一套保护色，把真实的自己藏起来。

提不起劲儿的“累觉不爱”： 第三象限里 DEAD(死者) 和 SOLO(独行者) 的流行，真的很像大家常说的“精神内耗”和“倦怠感”。当发现拼命努力也没啥用时，很多人干脆就不投入了，选择身心双重躺平。

情绪被掏空： 第四象限（卷入内耗型）那么挤，说明现在的年轻人普遍都经历着情绪上的严重透支啊！

写在最后

我们这次从死板的对应表升级到动态的热力图，不光是一次算法的升级，更是想好好聊聊“我们到底是怎么在这个社会里生活的”。

我们不想冷冰冰地告诉你“你就是个什么样的人”，而是想通过这些图表告诉你：“在这个复杂的世界里，作为独一无二的你，偶尔戴上不同的面具、呈现出千姿百态的碎片，都是极其正常且有迹可循的！” 这才是我们做这套系统最想传达给大家的。
